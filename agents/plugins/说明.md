插件是一组mcp或者skill的说明

使用方法
# 列出可用插件
python scripts/plugin-manager.py list

# 安装插件（模拟运行）
python scripts/plugin-manager.py install agents/plugins/computer-use.plugin.json --dry-run

# 真正安装插件
python scripts/plugin-manager.py install agents/plugins/computer-use.plugin.json
这样您就可以创建统一格式的插件配置文件，然后用一个命令自动安装，无需手动一个个配置了！



比如说computer-use插件

1. 配置mcp
```json
  "open-computer-use": {
   "name": "open-computer-use",
       "command": "npx",
       "args": [
  "-y",
 "open-computer-use", 
         "mcp"]
   }
 ```
2. 配置skill

npx skills add iFurySt/open-codex-computer-use  --skill open-computer-use -y
