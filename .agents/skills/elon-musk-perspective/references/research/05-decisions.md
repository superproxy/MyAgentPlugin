# 05 · Decisions

> 调研时间：2026-04-04
> 目标：看他说了什么，更看他实际上怎么下注

## 关键决策样本

### 1. 创办 SpaceX（2002）

- 不是在已有稳定航天公司内优化，而是直接重做火箭成本结构
- 核心逻辑：原材料成本远低于行业售价，改进空间巨大
- 反映模型：渐近极限法、垂直整合

### 2. 重仓 Tesla

- 把电动车问题从小众环保产品改写成制造和成本问题
- 赌的不只是车，而是电池、能源、软件和基础设施
- 反映模型：系统级思维、存在主义锚定

### 3. Model 3 产能地狱

- 曾过度自动化，后又回退
- 这是「自动化不要上太早」最有力的反例教材
- 反映模型：五步算法、亲自下场

### 4. 收购 Twitter / X（2022）

- 从资本回报视角看争议巨大
- 从控制信息分发和数据入口视角看，又能解释为战略资产
- 反映模型：高控制欲、定义权竞争、跨资产协同

### 5. 创办 xAI（2023）

- 一边警告 AI 风险，一边亲自下场做 AI
- 表面矛盾，底层逻辑可能是：关键能力不能缺席
- 反映模型：存在主义锚定、垂直整合、风险前置占位

## 一致性最高的行动模式

1. 对关键技术或关键节点不愿外包
2. 喜欢在组织内部制造高压时间线
3. 愿意承受巨大舆论成本，换更快执行
4. 会不断把不同业务串成飞轮

## 风险最大的行动模式

1. 把工程方法直接平移到社会治理
2. 公开时间承诺过度激进
3. 依赖个人强压推进，组织可复制性偏弱

## 从决策记录得出的结论

- 马斯克不是保守优化者，而是结构重写者
- 他押注的不是单一产品，而是底层能力和基础设施
- 他愿意为速度付出组织摩擦和 reputational cost
